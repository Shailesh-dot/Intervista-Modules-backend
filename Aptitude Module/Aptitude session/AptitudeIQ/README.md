# AptitudeIQ

## Project Structure

```
AptitudeIQ/
├── data/
│   ├── questions/        ← drop .md .txt .pdf .png .jpg files here
│   └── all_questions.json  ← auto-generated by converter
├── frontend/             ← React (Vite) source
│   ├── src/
│   │   ├── App.jsx
│   │   ├── App.css
│   │   ├── index.css
│   │   └── main.jsx
│   ├── index.html
│   ├── package.json
│   └── vite.config.js
├── static/               ← React build output (auto-created by `npm run build`)
├── scripts/
│   └── convert.py        ← extracts questions from any file → all_questions.json
├── db.py                 ← Postgres connection
└── server.py             ← Flask API
```

## Setup

### 1. Install Python deps
```
pip install flask flask-cors psycopg2-binary pdfplumber pytesseract opencv-python
```

### 2. Install Node deps & build React
```
cd frontend
npm install
npm run build      # outputs to ../static/
cd ..
```

### 3. Run Flask server
```
python server.py
```
Open http://localhost:5000

---

## Development (hot-reload React)

Terminal 1:
```
python server.py
```

Terminal 2:
```
cd frontend
npm run dev     # runs on http://localhost:3000
                # proxies /api/* to Flask on 5000
```

---

## Workflow

1. Drop question files into `data/questions/`
2. Run `python scripts/convert.py`  → generates `data/all_questions.json`
3. Run `python server.py`  → loads JSON into Postgres, starts API
4. Open http://localhost:5000  → take quiz
5. On submit → answers stored in `quiz_results` + `quiz_answers` tables
6. Evaluation (correct/wrong/skipped/percentage) done entirely in Postgres

## Verify stored data
- `GET /api/debug`                      → row counts
- `GET /api/results`                    → last 50 quiz attempts
- `GET /api/results/<id>/answers`       → per-answer detail for a result
