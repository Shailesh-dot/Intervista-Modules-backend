KEY FORMULAS:
Average = Sum of all values / Number of values
Sum = Average x Number of values
New avg after replacement = Old avg + (New value - Old value) / n

Q1. Average of 5 consecutive numbers is 15. Find the largest number.
A) 17   B) 18   C) 19   D) 16
Answer: A) 17
Solution: Middle=15. Numbers: 13,14,15,16,17. Largest=17.

Q2. Average of 20 numbers is 30. If 45 is replaced by 15, new average?
A) 28.5   B) 29   C) 27   D) 28
Answer: A) 28.5
Solution: New sum=600-45+15=570. New avg=570/20=28.5.
